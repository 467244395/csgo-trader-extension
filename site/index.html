CSGO Trader - Steam Trading Enhancer Extension

Hi, unfortunately the extension is not available at the Chrome Webstore. It's mostly Google's fault. I wrote this up to inform my users. I was not able to publish new updates for over a month (I had big new features ready like fade percentage checking and pricing), but it's been stuck in the status of "pending review" and Google's Dev support has been no help. Out of desperation I foolishly clicked "unpublish", which made it disappear from the Chrome Webstore and it turns out you can't re-publish while it's under "pending review" status... The last version that was available out there was 1.14.2, I have released 1.15 (which added fade percentage checking to inventories) and 1.16 (which added pricing information). This move definitely made me prioritize porting my extension to Firefox and other browsers. When it's (hopefully soon) back, it should be available where it used to in the Chrome Webstore.

For now you have install it manually. I will write up a more detailed and illustrated version of this guide, but I am on vacation so I have limited time:

  • Grab the Source code (zip) of the latest version from the Github Releases
  • Unzip it to a location you can find easily.
  • Open chrome://extensions/.
  • Turn on Developer mode on the top right.
  • Click "Load Unpacked".
  • Select the folder that has "manifest.json" right inside it.

The Extension should now install and you can use it the same way if you have installed it from the Webstore. Unfortunately you won't get extension updates this way so if you find it useful please check back here.
