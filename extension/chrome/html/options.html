CSGO Trader Options

General

Name Value Description
Steam API key Allows the extension to make API requests to the Steam Web API, functions that need an API key will be tagged with this icon:
You can get an API key by filling out this form (you can put anything in as domain name).
Tabs API access Grants the extension access to the browser.tabs API. It - for example - allows the extension to open a tab and put it in focus, allows notifications to be clickable, opens the bookmarks page when an item is bookmarked, etc. I am using it in this mode so it's also less likely to be buggy as well.
Flag scam comments Reports obvious scam and spam comments like "I will give my knife for all of your csgo graffitties" and others. Helps to keep steamcommunity cleaner and safer.
Your strings to report Make the extension report comments that includes one of the the strings you add here. These are additional to the built-in ones.
Colorful items Makes inventories and offers pretty by changing the background color of items based on rarity or in case of doppler, phase

Trade Offer

Name Value Description
Quick decline offers Clicking "decline trade" on the tradeoffers page declines the trade offer without the confirmation dialog popping up
Open trade window as a tab The trade window opens in a neat new tab instead of the annoying separate small window, especially useful when there is no mouse to middle-click (when on laptop or mobile)
Default sorting mode Specifies what method the items in an trade offer should be sorted by
Switch to other inventory When on, it makes the other party's inventory the active one by default when trade offers load
Get float values automatically Loads float values to each item when on

Name Value Description

Profile

Name Value Description
Show "+rep" button Puts a "+rep" button on profiles which when pressed comments your reputation message
Reputation message The content of the comment you want posted on someone's profile when the "+rep" button is pressed, defaults to simply "+rep"
Show "Reocc" button Puts a "Reocc" button on your own profile which when pressed comments your reoccuring message
Reoccuring message The content of the comment you want posted on your profile when the "Reocc" button is pressed, defaults to "I don't have other accounts. If someone adds you with my name and picture they are scammers."
NSFW When enabled, it removes profile backgrounds and artwork from profiles. Ideal for work and school environment - when you want to avoid anime boobs showing on your screen.
Show real status Shows users' real chat status on their profile - if they are away it will show away.
Mark scammers Changes background and adds warning ribbon to steamrep.com banned scammers' profile and trade offers they send.

Other

Name Value Description
Lounge auto-bump Automatically bumps your trades every 30-40 minutes on csgolounge.com if you have the page "https://csgolounge.com/mytrades" open. Note: Lounge is broken and it never disables the bump button, regardless trades do get bumped.
CSGOTraders.net auto-bump Automatically bumps your trades every 30-40 minutes on CSGOTraders.net if you have the page "https://csgotraders.net/mytrades" open. Note: The way it works looks the bump buttons unpressed, but it does the bumping as you can see if you refresh the page.
